Passing Data

You can pass data to your components by using the props property: 

const modalResult = this.$showModal({
    component: "modal-2",
    props: {
      age: {{ age }}
    }
});

And you can receive data from your modal by using the promise property of the ModalResult object: 

const modalResult = this.$showModal({
    component: "modal-2",
    props: {
      age: {{ age }}
    }
});

modalResult.promise
  .then(result => {
    //result is the value passed from your component calling this.$emit('close-modal', //some value);
  });

In your modal component you just need to call this.$emit('close-modal') to close your modal: 

//in your component's code ("modal-2" in this example)

{
  props: {
      age: {
        type: String,
        required: true
      }
  },
  data() {
    name : null
  },
  methods: {
    closeModal() {
      this.$emit('closeModal', {});
    }
  }
}
