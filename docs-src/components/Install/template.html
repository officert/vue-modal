Installation

npm install vue2-modal --save-dev

Usage

1) Install the Vue Modal plugin: 

import Vue from 'vue';
import VueModal from 'vue2-modal';

Vue.use(VueModal);

2) In your HTML add the component: 

<modal></modal>

3) Launch a modal from your component: 

methods: {
  showModal() {
    const modal1Handle = this.$showModal({
      component : 'your-component-name',
      props: {
        //any data you want passed to your component
      }
    })

    modal1Handle.promise
      .then(result => {

      });
  }
}

'your-component-name' just needs to be the name of a valid component that you have registered with Vue like this: Vue.component('your-component-name', {})

The $showModal() method returns a ModalResult object.

Learn more about the ModalResult object here.
