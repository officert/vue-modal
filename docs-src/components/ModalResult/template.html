Modal Result

The $showModal method returns a ModalResult object. 

methods: {
  showModal() {
    const modalResult = this.$showModal({
      component: 'your-component-name'
    });
  }
}

The ModalResult object allows you to show and hide the modal programatically.

Just store the ModalResult to your Vue component and use a method to call hide(). 

methods: {
  showModal() {
    this.modalResult = this.$showModal({
      component: 'your-component-name'
    });
  },
  hideModal() {
    this.modalResult.hide();
  }
}

You can also call show() to re-show a modal. 

methods: {
  showModal() {
    if (this.modalResult) {
      this.modalResult.show();

      return;
    }

    this.modalResult = this.$showModal({
      component: 'your-component-name'
    });
  }
}

The ModalResult object also contains a promise property, which is a Promise.

You can use this promise to receive data from your modal component. 

methods: {
  showModal() {
    const modalResult = this.$showModal({
      component: 'your-component-name'
    });

    modalResult.promise
      .then(result => {
        //result is whatever you're modal component passes to this.$emit("closeModal", {})
      });
  }
}
